# Прогресс разработки: Arkanoid (Rust + Bevy 0.18)

## ✅ Этап 0: Подготовка проекта
- [x] Rust toolchain (stable)
- [x] Создан проект (`Cargo.toml`, `bevy = "0.18"`, `rand = "0.8"`)
- [x] Настроено окно 800×600, фон — тёмно-синий
- [x] Пустая сцена запускается

## ✅ Этап 1: Базовая сцена и сущности
- [x] 2D камера (`Camera2d`)
- [x] Ракетка (Paddle) — синий прямоугольник внизу экрана (y = -250)
- [x] Мяч (Ball) — белый круг (r = 10)
- [x] Стены: левая, правая, верхняя (нижней нет — зона проигрыша)

## ✅ Этап 2: Управление игроком
- [x] A/D и ←/→ — движение ракетки (400 px/s)
- [x] Границы вычисляются динамически из `Collider.half_width` (корректно при PaddleGrow)

## ✅ Этап 3: Движение мяча
- [x] Компонент `Velocity { x, y }`
- [x] `apply_velocity_system` с `dt.min(0.05)` против туннелирования
- [x] Мяч стартует прилипшим к ракетке (`BallStuck`)
- [x] Пробел или движение ракетки — запуск мяча

## ✅ Этап 4: Коллизии (AABB)
- [x] Ball ↔ Wall — отскок с корректировкой позиции
- [x] Ball ↔ Paddle — угловой эффект, `hit_factor.clamp(-0.8, 0.8)`, мин. vy = speed×0.4
- [x] Нормировка скорости после каждого отскока

## ✅ Этап 5: Блоки
- [x] Сетка блоков из конфига уровня (`grid: &[&[u8]]`)
- [x] Normal (1 удар, 100 очков), Strong (2 удара, 200 очков)
- [x] 30% шанс дропа бонуса при уничтожении блока

## ✅ Этап 6: Игровая логика
- [x] Мяч y < −320 → потеря жизни, мяч → BallStuck над ракеткой
- [x] 0 жизней → `GameOver`
- [x] Все блоки уничтожены → `LevelComplete`
- [x] Enter/Space: рестарт из GameOver, LevelComplete
- [x] `LevelEntity` маркер + `cleanup_level` при `OnExit(Playing)`

## ✅ Этап 7: Система уровней
- [x] 5 уровней с разными паттернами блоков
- [x] `BallSpeedMultiplier`: нарастает по уровням
- [x] `CurrentLevel` ресурс, wraparound после последнего уровня

## ✅ Этап 8: Бонусы (Power-ups)
- [x] Дроп из блоков — 30% шанс, 7 типов
- [x] PaddleGrow — ракетка ×1.5 на 10 сек
- [x] StickyPaddle — мяч прилипает при касании на 10 сек
- [x] BallGrow — мяч ×1.5 на 10 сек
- [x] GunPaddle — пулемёт на 15 сек
- [x] FireBall — пробивает блоки насквозь на 8 сек

## ✅ Этап 9: НЛО (UFO)

## Управление (Playing)

| Клавиша | Действие |
|---------|----------|
| A / ← | ракетка влево |
| D / → | ракетка вправо |
| Пробел | запуск мяча |
| Ctrl (Left/Right) | стрельба пулемётом |
| Escape | пауза; повторно → MainMenu |
| F2 | музыка вкл/выкл |
| `*` Numpad | **[DEBUG]** следующий уровень |

## Управление (GameOver)

| Клавиша | Действие |
|---------|----------|
| Enter / Space | рестарт с первого уровня |
| ESC | выход в MainMenu (сброс счёта/уровня) |

## Управление (HighScores)

| Клавиша | Действие |
|---------|----------|
| Enter / ESC | вернуться в MainMenu |

## Управление (EnterName)

| Клавиша | Действие |
|---------|----------|
| Буквы / цифры | ввод имени (макс. 12 символов) |
| Backspace | удалить последний символ |
| Enter | подтвердить имя и сохранить рекорд |

## Управление (Options)

| Клавиша | Действие |
|---------|----------|
| W / S или ↑ / ↓ | навигация |
| ← / → | изменить громкость ±10% |
| ESC | вернуться в MainMenu |

## Управление (LevelEditor)

| Клавиша | Действие |
|---------|----------|
| ЛКМ / drag | рисовать кистью |
| ПКМ / drag | стирать |
| 1–6 | выбрать цвет (Blue/Green/Yellow/Orange/Purple/Red) |
| T | переключить тип Normal ↔ Strong |
| 0 | кисть-стёрка |
| ← / → | предыдущий / следующий уровень |
| N | создать новый пустой уровень |
| + / - | добавить/убрать ряд |
| S / L | сохранить / загрузить (`levels/level_N.lvl`) |
| P | играть редактируемый уровень |
| Escape | в главное меню |
