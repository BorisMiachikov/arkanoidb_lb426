# Roadmap разработки: Breakout (Rust + Bevy)

## Этап 0: Подготовка проекта

**Цель:** Создать базовую структуру и убедиться, что окружение работает.

### Задачи:

* [ ] Установить Rust toolchain (stable)
* [ ] Создать новый проект (`cargo new`)
* [ ] Подключить Bevy 0.15
* [ ] Настроить запуск окна
* [ ] Проверить рендер пустой сцены

### Результат:

Минимальное приложение с окном Bevy.

---

## Этап 1: Базовая сцена и сущности

**Цель:** Отобразить основные игровые объекты.

### Задачи:

* [ ] Добавить камеру (2D)
* [ ] Создать ракетку (Paddle)
* [ ] Создать мяч (Ball)
* [ ] Добавить стены (границы экрана)
* [ ] Визуализировать объекты (спрайты или простые формы)
