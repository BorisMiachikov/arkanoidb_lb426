# Roadmap разработки: Arkanoid (Rust + Bevy 0.15)

## ✅ Этап 0: Подготовка проекта

**Цель:** Создать базовую структуру и убедиться, что окружение работает.

### Задачи:

* [x] Установить Rust toolchain (stable)
* [x] Создать новый проект (`cargo new`, `bevy = "0.15"`)
* [x] Настроить запуск окна 800×600
* [x] Проверить рендер пустой сцены

### Результат:

Минимальное приложение с окном Bevy.

---

## ✅ Этап 1: Базовая сцена и сущности

**Цель:** Отобразить основные игровые объекты.

### Задачи:

* [x] Добавить камеру (2D)
* [x] Создать ракетку (Paddle)
* [x] Создать мяч (Ball)
* [x] Добавить стены (левая, правая, верхняя)
* [x] Визуализировать объекты (`Mesh2d` + `MeshMaterial2d`)

### Результат:

На экране отображаются ракетка, мяч и границы.

**Цель:** Сделать управление ракеткой.

### Задачи:

* [x] Обработать ввод с клавиатуры (A/D или ←/→)
* [x] Ограничить движение границами экрана (без наезда на стены)
* [x] Настроить скорость движения (400 px/s)

### Результат:

Игрок может управлять ракеткой.

## ✅ Этап 3: Движение мяча

**Цель:** Реализовать базовую кинематику мяча.

### Задачи:

* [x] Добавить компонент `Velocity`
* [x] Реализовать систему движения (`apply_velocity_system`)
* [x] Мяч стартует прилипшим к ракетке (`BallStuck`)
* [x] Запуск мяча пробелом или движением ракетки

### Результат:

Мяч движется по экрану, запускается от ракетки.

---

## ✅ Этап 4: Коллизии (AABB)

**Цель:** Реализовать базовую физику.

### Задачи:

* [x] Реализовать AABB-коллизии (`aabb_collision`)
* [x] Ball ↔ Wall — отскок с корректировкой позиции
* [x] Ball ↔ Paddle — угловой эффект, `hit_factor.clamp(-0.8, 0.8)`
* [x] Минимальная вертикальная скорость после отскока

### Результат:

Мяч корректно отскакивает от стен и ракетки.

---

## ✅ Этап 5: Блоки (Bricks)

**Цель:** Добавить разрушаемые объекты.

### Задачи:

* [x] Создать сущность `Brick { brick_type, health, score_value }`
* [x] Генерация сетки блоков из конфига уровня
* [x] Коллизии Ball ↔ Brick
* [x] Типы: Normal (1 удар, 100 очков), Strong (2 удара, 200 очков)
* [x] Удаление блока при `health == 0`

### Результат:

Мяч разрушает блоки.

---

## ✅ Этап 6: Игровая логика

**Цель:** Ввести базовые правила игры.

### Задачи:

* [x] Отслеживание падения мяча за нижнюю границу
* [x] Система жизней (Lives, default: 3)
* [x] При потере жизни — мяч прилипает к ракетке (BallStuck)
* [x] При 0 жизней — `GameOver`
* [x] Проверка победы (все блоки уничтожены → `LevelComplete`)
* [x] Enter/Space: рестарт из GameOver и LevelComplete

### Результат:

Игра имеет условия победы и поражения.

---

## ✅ Этап 7: Система уровней

**Цель:** Добавить прогрессию.

### Задачи:

* [x] Формат описания уровня (`LevelConfig` — статические данные в `level_data.rs`)
* [x] Загрузка уровня по индексу `CurrentLevel`
* [x] Переход между уровнями (wraparound после последнего)
