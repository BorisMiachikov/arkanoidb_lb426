# Техническое задание: Игра в жанре Arkanoid/Breakout (Rust + Bevy 0.15)

## 1. Общая информация

**Название проекта:** arkanoidb_lb426
**Жанр:** Аркада / Breakout
**Язык разработки:** Rust (stable)
**Движок:** Bevy 0.15
**Физика:** Кастомная (AABB-коллизии, без Rapier)
**Репозиторий:** https://github.com/BorisMiachikov/arkanoidb_lb426

---

## 2. Описание игры

Игрок управляет платформой (ракеткой) в нижней части экрана.
Цель — отбивать мяч и уничтожать все блоки на уровне.

Игра заканчивается:
- **Победой** — при уничтожении всех блоков → `LevelComplete`
- **Поражением** — если мяч падает за нижнюю границу при 0 жизнях → `GameOver`

---

## 3. Основной игровой цикл

1. Инициализация уровня (`OnEnter(Playing)` → `spawn_level_entities`)
2. Мяч появляется прилипшим к ракетке (`BallStuck`)
3. Запуск мяча пробелом или движением ракетки
4. Обработка ввода → движение → коллизии → эффекты
5. Проверка условий победы/поражения
6. Переход к следующему уровню или рестарт

---

## 4. Игровые сущности

### 4.1 Ракетка (Paddle)
- Управляется: A/D или ←/→, скорость 400 px/s
- Ограничена стенами: x ∈ [−324, 324]
- Эффекты: PaddleGrow (×1.5 на 10 сек), StickyPaddle (прилипание на 10 сек)

### 4.2 Мяч (Ball)
- Размер: 20×20, начальная скорость: vx=200, vy=350 (×`BallSpeedMultiplier`)
- Отскакивает от: стен, ракетки, блоков, НЛО
- Угол отскока от ракетки зависит от точки контакта
- Эффект: BallGrow (×1.5 на 10 сек)

### 4.3 Блоки (Bricks)
- Normal: 1 удар, 100 очков
- Strong: 2 удара, 200 очков (красный)
- 30% шанс дропа бонуса при уничтожении

### 4.4 Бонусы (Power-ups)

| Тип | Эффект | Длительность |
|-----|--------|-------------|
| PaddleGrow | ракетка ×1.5 | 10 сек |
| StickyPaddle | мяч прилипает | 10 сек |
| BallGrow | мяч ×1.5 | 10 сек |
| GunPaddle | пулемёт | *(не реализован)* |

Поведение: падают вниз, активируются при касании ракеткой.

### 4.5 НЛО (UFO)
- Движутся горизонтально, отражаются от стен
- Уничтожаются за **2 удара** мячом
- После уничтожения **респавнятся на случайной высоте** (y ∈ [−120, 40])
- Конфигурируются в `LevelConfig`: позиция, скорость, интервал бомб

### 4.6 Бомбы (Bomb)
- Сбрасываются НЛО по таймеру, падают со скоростью −220 px/s
- При попадании в **ракетку** → потеря жизни (или GameOver)
- При попадании в **блок** → бомба исчезает (блок не повреждается)
- Исчезают за нижней границей экрана (y < −350)

## 5. Уровни

Данные уровней — статический массив `LEVELS` в `src/resources/level_data.rs`:

| Уровень | Паттерн | НЛО | Скорость | Бомба (интервал) |
|---------|---------|-----|---------|-----------------|
| 1 | Классический (10×5) | 0 | ×1.0 | — |
| 2 | Шахматный | 1 | ×1.25 | 4 сек |
| 3 | Пирамида | 2 | ×1.5 | 3 сек |

Формат сетки: `&[&[u8]]`, где `0` = пусто, `1` = Normal, `2` = Strong.

---

## 6. Физика и коллизии

Подход: **AABB** (Axis-Aligned Bounding Box).

Проверяемые пары:

| Пара | Эффект |
|------|--------|
| Ball ↔ Wall | отскок |
| Ball ↔ Paddle | угловой отскок |
| Ball ↔ Brick | отскок + урон блоку |
| Ball ↔ UFO | отскок + урон НЛО |
| Bonus ↔ Paddle | подбор бонуса |
| Bomb ↔ Paddle | потеря жизни |
| Bomb ↔ Brick | бомба исчезает |

---

## 7. Управление

| Клавиша | Действие |
|---------|----------|
| A / ← | ракетка влево |
| D / → | ракетка вправо |
| Пробел | запуск мяча / прилипший мяч |
| `*` Numpad | **[DEBUG]** следующий уровень |

---

## 8. Состояния игры (Game States)

```
Startup → Playing ↔ LevelComplete → Playing (следующий уровень)
                 ↘ GameOver → Playing (рестарт)
```

| Состояние | Описание |
|-----------|----------|
| `MainMenu` | Главное меню *(Этап 12)* |
| `Playing` | Активная игра |
| `Paused` | Пауза *(Этап 12)* |
| `LevelComplete` | Уровень пройден |
| `GameOver` | Игра окончена |

## 9. Архитектура (Bevy ECS)

### Плагины

| Плагин | Ответственность |
|--------|----------------|
| `GameplayPlugin` | Правила игры, ресурсы, победа/поражение |
| `PhysicsPlugin` | Ввод, движение, все коллизии, бонусы, НЛО |
| `UiPlugin` | Интерфейс *(Этап 11)* |
| `LevelPlugin` | Камера, спавн/очистка уровня |

### Компоненты (только данные)

```
Velocity { x, y }
Collider { half_width, half_height }
Paddle { speed }
Ball { radius }  +  BallStuck (маркер)
Brick { brick_type, health, score_value }
Bonus { bonus_type }
PaddleGrowEffect / StickyEffect / BallGrowEffect { timer, ... }
Ufo { speed, direction, bomb_timer, health }
Bomb { damage }
Wall (маркер)
LevelEntity (маркер для cleanup)
```

### Ресурсы

```
GameState (States enum)
Score { value }
Lives { count }
CurrentLevel { number }
BallSpeedMultiplier(f32)
DebugSkipPending(bool)
```

### Структура `src/`

```
src/
├── main.rs
├── app.rs
├── plugins/       gameplay, physics, ui, level
├── components/    ball, brick, bonus, bonus_effects, bomb, collider,
│                  level_entity, paddle, ufo, velocity, wall
├── systems/       input, movement, collision, gameplay, bonus, ufo
├── resources/     game_state, score, level_data
└── setup/         camera, level
```
