# Техническое задание: Игра в жанре Breakout (Rust + Bevy)

## 1. Общая информация

**Название проекта:** Breakout Clone (рабочее название)  
**Жанр:** Аркада / Breakout  
**Язык разработки:** Rust  
**Движок:** Bevy (версия 0.15)  
**Физика:** Кастомная (AABB-коллизии)

---

## 2. Описание игры

Игрок управляет платформой (ракеткой), расположенной в нижней части экрана.  
Основная задача — отбивать мяч и уничтожать блоки на уровне.

Игра заканчивается:
- Победой — при уничтожении всех блоков
- Поражением — если мяч падает за нижнюю границу экрана

---

## 3. Основной игровой цикл

1. Инициализация уровня
2. Запуск мяча
3. Обработка ввода игрока
4. Обновление позиций объектов
5. Проверка коллизий
6. Применение эффектов (бонусы, ускорение)
7. Проверка условий победы/поражения
8. Переход к следующему уровню или рестарт

---

## 4. Игровые сущности (Entities)

### 4.1 Ракетка (Paddle)
- Управляется игроком (влево/вправо)
- Ограничена границами экрана
- Может изменять размер
- Может иметь эффекты:
  - Липкость (мяч прилипает)
  - Оружие (пулемет)

### 4.2 Мяч (Ball)
- Движется с постоянной скоростью
- Отражается от:
  - Ракетки
  - Стен
  - Блоков
  - НЛО
- Увеличивает скорость со временем
- Может менять размер

### 4.3 Блоки (Bricks)
- Разрушаются при столкновении с мячом
- Возможны типы:
  - Обычные (1 удар)
  - Прочные (несколько ударов)
- Могут дропать бонусы

### 4.4 Бонусы (Power-ups)
Типы:
- Увеличение ракетки
- Липкая ракетка
- Пулемет
- Увеличение мяча

Поведение:
- Падают вниз
- Активируются при столкновении с ракеткой
- Имеют длительность действия (если применимо)

### 4.5 НЛО (UFO / NPC)
- Двигаются по уровню (горизонтально или по паттерну)
- Являются препятствием для мяча
- Могут:
  - Отражать мяч
  - Сбрасывать бомбы

### 4.6 Бомбы
- Падают сверху вниз
- При столкновении с ракеткой:
  - Наносят урон / уменьшают жизнь / штраф

---

## 5. Уровни

### Требования:
- Поддержка нескольких уровней
- Уровень описывается данными (желательно JSON/ron/ресурс)
- Параметры уровня:
  - Расположение блоков
  - Типы блоков
  - Количество НЛО
  - Начальная скорость мяча

### Прогрессия:
- Увеличение сложности:
  - Скорость мяча
  - Количество блоков
  - Частота появления НЛО
  - Частота бонусов

---

## 6. Физика и коллизии

### Подход:
- Axis-Aligned Bounding Box (AABB)

### Требования:
- Проверка столкновений:
  - Ball ↔ Paddle
  - Ball ↔ Brick
  - Ball ↔ Wall
  - Ball ↔ UFO
  - Bonus ↔ Paddle
  - Bomb ↔ Paddle

### Поведение:
- Отражение мяча:
  - Изменение направления по нормали
  - Угол отражения зависит от точки контакта с ракеткой

---

## 7. Управление

- Движение ракетки:
  - Клавиатура (A/D или ←/→)
- Дополнительно:
  - Стрельба (если активен пулемет)

	Gameplay → правила игры
